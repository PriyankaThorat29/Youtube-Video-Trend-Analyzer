# Youtube-Video-Trend-Analyzer
This project analyzes YouTube video data to find which categories perform better based on views, likes, and engagement. It uses basic Python concepts like lists, loops, and calculations to generate insights.
YouTube Video Trend Analyzer is a data science project that analyzes YouTube video data to find which categories perform best based on views, likes, and engagement. It uses Python to process data, calculate performance metrics, and identify popular content trends. This project helps understand digital media performance and audience preferences.
